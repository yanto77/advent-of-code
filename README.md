# advent-of-code

Solutions for [Advent of Code](http://adventofcode.com/) in C++.

Project is based on [Voltara/advent2018-fast](https://github.com/Voltara/advent2018-fast), but has been modified since.

| **Year** | **Stars** | **Runtime** |
|:--------:|:---------:|------------:|
|   2019   |    26*    |       58 ms |
|   2020   |    50*    |      660 ms |
|   2021   |    23*    |        2 ms |
|   2022   |    8*     |        0 ms |
|  Total   |   107*    |  720-750 ms |
